# 2026dsa_finalproject_group10
## About
This project predicts corn hybrid yield using machine learning models trained on variety trial data from 2014-2023 across 45 US sites across the USA.
## Team
- Amit Godara (Student 1)
- Taiwo Owolanke (Student 2)

## ModelsXGBoost
## GLMNet Elastic Net

## Data
- Training: 164,477 rows, 2014-2023
- Testing: 10,057 rows, 2024
- Weather data pulled from NASA POWER API using nasapower package

## Repository Structure
- code/ - R scripts and Quarto files
- data/ - training and testing data
- output/ - final predictions and plots

## Goal
Predict 2024 corn corn yield (mg/ha) for 23 sites across the USA
